# vue-sushi
Адаптив до 320px.
Реализовано: 
Каталог товаров, краткое описание товара, добавление в корзину товара и удаление, количество и общая стоимость, поиск товара по имени, push уведомление при добавлении в корзину.  
## Project setup
```
npm install
```
